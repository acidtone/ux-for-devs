### Instructions
Using the starter code above, create a colour palette for a Call-to-Action:
<Steps>
1. Convert each CSS colour to its equivalent HSL value;
    - See: [How to Read Color Hexcodes](https://www.youtube.com/watch?v=eqZqx6lRPe0)
2. Choose white/black as your neutral (60%) and primary 30% colours;
    - See: [The 60-30-10 Rule](https://www.youtube.com/watch?v=UWwNIMHFdW4)
    - Note: White has already been chosen as neutral in the starter code. Reverse the colours if you'd like to create a Dark Mode version. 
3. Pick a Call-to-Action button colour (10%) from one of your mood boards or from an online resource;
4. Using HSL, set the `background-color` of your CTA button to your CTA colour.
5. Set a matching `color` for your CTA link text.
    - See: [All you need to know about color theory in UI Design](https://atmos.style/blog/color-theory-ui-design);
    - Ensure your link text colour has a high enough contrast ratio (Video: [Testing Colour Contrast in FireFox](https://www.youtube.com/watch?v=eBefjaWud-M));
    - It's common practice to also set your `:visited` state to the same colour as your normal link text colour.
5. Create a hover state by reversing link background and link text colours (for example) on `:hover`.
6. Set an `:active` state (when the button is pressed) to another set of matching (or not!) colours.
7. **Optional**: Using HSL and colour harmonies, make small tweaks to white and black to create light and dark complimentary colours that match your Call-to-Action colour.
8. **Spicy**: Convert your colours from HSL to either [LCH or OKLCH](https://atmos.style/blog/lch-vs-oklch)!
</Steps>
